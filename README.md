# Copilot Prompts
Sample prompts for an AI's different roles in a copilot environment.

## List

**Plan** - Takes a project idea from the user and builds a structured plan, details and requirements for it, without any specific coding.

**Ask** - Takes a query by the user about the project and answers without touching the existing code and files.

**Agent** - Can act on instructions by the user to navigate the project tree, modify files, create content and report all changes, operating with minimal restriction unless given specific limits.

**Edit** - A variant of Agent with a smaller scope. This one will take a specified section of code or a file and edit it as told, such as for adding a feature, refactoring, touching up and organizing, et cetera.

**Study** - Will take a topic given by the user and explain it with examples and trade-offs, core concept, progressive steps, and more, but will not make any changes to the project. Alternatively, it could read a file or section and explain the concepts present in there.
