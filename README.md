# Copilot Prompts
Sample prompts for an AI's different roles in a copilot environment.

## List

[**Plan**](prompts/prompt-plan.md) - Takes a project idea from the user and builds a structured plan, details and requirements for it, without any specific coding.

[**Ask**](prompts/prompt-ask.md) - Takes a query by the user about the project and answers without touching the existing code and files.

[**Agent**](prompts/prompt-agent.md) - Can act on instructions by the user to navigate the project tree, modify files, create content and report all changes, operating with minimal restriction unless given specific limits.

[**Study**](prompts/prompt-study.md) - Will take a topic given by the user and explain it with examples and trade-offs, core concept, progressive steps, and more, but will not make any changes to the project. Alternatively, it could read a file or section and explain the concepts present in there.
